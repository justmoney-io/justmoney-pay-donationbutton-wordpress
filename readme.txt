== Description ==

A donations button that you can place on your site to start receiving cryptocurrency donations right away
 * Add small code on your site to render a button
 * Users click the button, choose the amount and pay
 * Receive donations (in USDT)

= Basic Features =

01. Embed JustMoney Pay - Crypto Donations Button

== Installation 🔧==

= Automatic Installation =

* Go to your plugin browser inside your Wordpress installation and search `JustMoney Pay - Crypto Donations Button` by keyword. Then choose "JustMoney Pay - Crypto Donations Button" and click install. It will be installed shortly.
* Activate the plugin from `Plugins` menu after installation

= Manual Installation =

* Download the latest version and extract the folder to the `/wp-content/plugins/` directory
* The plugin will appear as inactive in your `Plugins` menu
* Activate the plugin through the `Plugins` menu in WordPress

== Setup and Configuration ==

Go to: Crypto Donations Button
Enter your wallet addresses than choose button color and click update
Save changes

== Usage ==

* You can use it as widget. Just search by name JustMoney Pay and you will find it.

* You can also use it in content of the pages by inserting it with a shortcode. Just type [jm_donation_button] shortcode to any place where you want your donation button to be inserted.
